# DBSAFER POC 標準作業流程

此專案為互動式 POC 流程網頁，使用 Tailwind CSS + Chart.js，
可部署於 GitHub Pages、Netlify、Vercel 等靜態託管平台。

## 檔案說明

- `index.html`：主頁，引用 `style.css`、`script.js`。
- `style.css`：自訂版面配置與流程樣式。
- `script.js`：Chart.js 圖表邏輯。
- `README.md`：使用與部署說明。
